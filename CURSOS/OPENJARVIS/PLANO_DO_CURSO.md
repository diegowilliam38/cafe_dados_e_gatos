# Curso OpenJarvis

## Objetivo do curso

Ensinar, de forma prática e progressiva, como entender, preparar e testar o OpenJarvis usando os caminhos principais do projeto:

- CLI
- Browser App
- Desktop App

O curso tem foco em teste real de ferramenta, documentação oficial e aprendizado prático.

## Resultado esperado

Ao final do curso, a pessoa será capaz de:

- entender a proposta do OpenJarvis;
- reconhecer os principais componentes da arquitetura;
- explicar os cinco pilares do projeto;
- preparar um ambiente local para testes;
- compreender o papel do Ollama e dos modelos locais;
- testar o OpenJarvis pela CLI;
- testar o OpenJarvis pelo Browser App;
- testar o OpenJarvis pelo Desktop App;
- validar se cada teste funcionou.

## Público-alvo

- pessoas interessadas em IA local;
- usuários de Ollama;
- estudantes de Ciência de Dados;
- desenvolvedores;
- criadores de conteúdo técnico;
- pessoas interessadas em agentes de IA.

## Estrutura oficial do curso

### Aula 1 — Como entender, preparar e testar o OpenJarvis pela CLI

Objetivo: apresentar o OpenJarvis, explicar sua proposta, entender a arquitetura principal e preparar o ambiente para o primeiro teste pela CLI.

### Aula 2 — Como testar o OpenJarvis no Browser App

Objetivo: testar o OpenJarvis usando a interface web do projeto.

### Aula 3 — Como testar o OpenJarvis no Desktop App

Objetivo: testar o OpenJarvis usando o aplicativo desktop do projeto.

## Critérios de qualidade

- O curso possui pelo menos 3 aulas.
- Cada aula ensina uma habilidade prática.
- Cada aula possui objetivo claro.
- Cada aula possui exercício.
- Cada aula possui checklist de validação.
- O conteúdo mantém foco educacional, técnico e prático.
- O foco permanece no OpenJarvis.
- Não há comparação com o ecossistema Frankenstein de IAs.
