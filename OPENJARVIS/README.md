Documentação e testes práticos do projeto OpenJarvis, do Stanford Scaling Intelligence Lab, para o canal Café com Dados & Gatos.

## Objetivo

Organizar os testes do OpenJarvis por grau de evolução:

- CLI no Linux
- Browser App
- Desktop App
