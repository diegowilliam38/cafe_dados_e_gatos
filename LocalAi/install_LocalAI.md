# Como instalar o LocalAI no Windows com WSL2 + Docker e testar LLM local

## 1. Objetivo

Rodar o LocalAI no Windows usando WSL2 + Docker Desktop e validar a instalação com um modelo LLM local leve.

Neste guia vamos usar:

- LocalAI
- Docker Desktop
- WSL2 com Ubuntu
- dolphin3.0-llama3.2-1b
- CPU-only

## 2. Ambiente deste guia

Este guia é para:

- Windows 10 ou Windows 11.
- WSL2 instalado.
- Ubuntu instalado dentro do WSL2.
- Docker Desktop instalado no Windows.
- Integração do Docker Desktop com WSL2 ativada.

Todos os comandos deste guia devem ser executados dentro do Ubuntu/WSL, não no PowerShell.
