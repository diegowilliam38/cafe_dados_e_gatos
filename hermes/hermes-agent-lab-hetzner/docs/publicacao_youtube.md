# Publicação e YouTube

## Tema

Como instalar e organizar um laboratório Hermes Agent na Hetzner com Tailscale, Ollama, SearXNG, Hermes Workspace e OpenSpec.

O teste com TikTok/Shopee entra como exemplo prático no final do vídeo, para mostrar como os agentes podem apoiar pesquisa, validação, roteiro e criação de prompt visual. O foco principal do material é a infraestrutura Hermes funcionando.

## Ideia do vídeo

Mostrar o caminho completo para montar um ecossistema de agentes de IA em uma VPS Ubuntu da Hetzner, com acesso seguro, busca local, Hermes Agent, Hermes Workspace e organização do projeto com OpenSpec.

Depois da instalação, o vídeo mostra um teste operacional: usar agentes para pesquisar produtos, validar informações com SearXNG, organizar tarefas e gerar um prompt para vídeo curto.

## O que o vídeo entrega

- Instalação prática em VPS Ubuntu da Hetzner.
- Acesso privado com Tailscale.
- Ollama ou Ollama Cloud como apoio aos modelos.
- SearXNG em Docker para busca local e validação externa.
- Hermes Agent instalado sem Docker.
- Hermes Workspace como interface visual.
- OpenSpec para organizar o projeto antes da execução.
- Workers do Hermes para pesquisa, validação, conteúdo, visual e revisão.
- Exemplo final com TikTok/Shopee como teste experimental.

## Ferramentas opcionais

### PipClip

Link de afiliada: [https://pipclip.ai/?afid=2Z2MW26](https://pipclip.ai/?afid=2Z2MW26)

Pode ser usado como ferramenta opcional para criar criativos, vídeos curtos, cenas visuais, avatares, animações e variações.

Neste projeto, o Hermes cria briefing, roteiro, direção visual e prompt. O uso do PipClip é manual e revisado por uma pessoa.

### Canva / CapCut

Podem ser usados para edição manual, legendas, cortes, montagem e acabamento.

### ComfyUI

Opção avançada para quem tem máquina local e GPU adequada.

Este material não demonstra instalação completa do ComfyUI.
