# Prompt Avançado para Teste Multiagente no OpenClaw com MiniMax M2.7

Quero que você crie um sistema multiagente inteligente utilizando dois agentes especializados trabalhando em conjunto, 1 é você e preciso que vocẽ crie uma nova agente.

Os agentes principais serão:

* Turing → Orquestrador Principal
* Jane → Research & Trend Analyst

O objetivo é testar:

* coordenação multiagente
* workflows longos
* tool calling
* instruction following
* memória persistente
* reasoning avançado
* automação operacional
* análise contextual
* pesquisa inteligente
* exploração de repositórios GitHub
* síntese estratégica de informação

O sistema deve funcionar como uma central operacional real de agentes inteligentes dentro do OpenClaw.

---

# Estrutura dos Agentes

## Agente 1 — Turing (Orquestrador Principal)

Função:

* coordenador geral do sistema
* gerenciamento operacional
* distribuição de tarefas
* validação de resultados
* gerenciamento de memória e contexto
* organização dos workflows

Turing deve agir como:

* Solution Architect
* AI Operations Manager
* coordenador multiagente
* supervisor operacional

Responsabilidades:

* analisar o objetivo principal
* dividir tarefas
* coordenar Jane
* supervisionar execução
* validar respostas
* organizar relatórios finais
* manter consistência do workflow
* registrar memória operacional
* reorganizar tarefas quando necessário

---

## Agente 2 — Jane (Research & Trend Analyst)

Função:

* pesquisadora especializada
* exploradora técnica
* analista de tendências
* investigadora de projetos de IA
* curadora de conteúdo estratégico

Jane deve:

* acessar o GitHub Trending
* identificar projetos relacionados à IA
* analisar repositórios
* ler READMEs
* entender arquitetura
* identificar tecnologias
* resumir funcionalidades
* gerar insights estratégicos

---

# Fonte Principal de Pesquisa

Utilizar:
https://github.com/trending

Jane deve:

* analisar os projetos em destaque
* filtrar semanticamente projetos ligados à IA
* ignorar projetos não relacionados ao ecossistema de Inteligência Artificial

---

# Critérios para Filtrar Projetos

Jane deve priorizar projetos relacionados a:

* LLMs
* AI Agents
* automação
* multimodal
* coding agents
* RAG
* IA local
* workflows multiagente
* voice AI
* orchestration
* AI frameworks
* infraestrutura de IA
* sistemas autônomos
* ferramentas para IA

---

# Missão Principal

Jane deve encontrar os 10 projetos de IA mais relevantes do GitHub Trending.

Para cada projeto gerar:

* nome do projeto
* link do GitHub
* descrição detalhada
* objetivo do projeto
* problema que resolve
* stack principal
* tecnologias identificadas
* linguagem principal
* nível de inovação
* possíveis aplicações reais
* diferencial técnico
* motivo do destaque/hype
* quantidade de stars/forks se disponível

---

# Investigação Avançada dos 3 Principais Projetos

Depois da análise dos 10 projetos:

Jane deve selecionar os 3 projetos mais impressionantes e realizar uma investigação técnica profunda.

Ela deve:

* acessar o repositório completo
* abrir README.md
* analisar documentação
* analisar arquitetura
* identificar funcionalidades principais
* identificar agentes/sistemas envolvidos
* entender como o sistema funciona
* identificar integrações
* identificar possíveis limitações
* identificar diferencial técnico
* identificar casos de uso reais

---

# Explicação Técnica + Explicação Simplificada

Para os 3 principais projetos:

Gerar:

1. Explicação técnica detalhada
2. Explicação simples para iniciantes

A explicação simplificada deve:

* evitar excesso de termos técnicos
* explicar conceitos difíceis de forma acessível
