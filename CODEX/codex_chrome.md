# Como usar o Codex com Chrome no Windows

## 1. Objetivo

Instalar e testar o Codex no Windows com a extensão do Chrome.

A ideia é permitir que o Codex use o navegador Chrome real para tarefas que precisam de sessão logada, abas, cookies e sites já acessados.

---

## 2. Cuidados antes de começar

Não use o Chrome principal.

Crie um perfil separado no Chrome apenas para testes com IA.

Evite deixar aberto ou logado:

- banco
- WhatsApp pessoal
- e-mail principal
- painel financeiro
- contas sensíveis
- senhas salvas importantes

---

## 3. Criar perfil separado no Chrome

Abra o Chrome.

Clique no ícone do perfil.

Clique em:

Adicionar perfil

Nome sugerido:

Codex-Teste

Use esse perfil para os testes do vídeo.

---

## 4. Instalar o Codex no Windows

Opção 1: instalar pela Microsoft Store.

Abra a Microsoft Store.

Pesquise por:

Codex

Instale o aplicativo oficial.

Opção 2: instalar via terminal do Windows:

winget install Codex -s msstore

---

## 5. Abrir o Codex

Abra o aplicativo Codex no Windows.

Faça login com sua conta ChatGPT/OpenAI.

---

## 6. Ativar o plugin do Chrome

No Codex, abra:

Plugins

Adicione o plugin:

Chrome

Siga o fluxo de instalação.

O Codex deve orientar a instalação/conexão da extensão no Chrome.

---

## 7. Confirmar conexão no Chrome

Abra o Chrome no perfil:

Codex-Teste

Verifique a extensão do Codex.

Ela deve aparecer como:

Connected

---

## 8. Primeiro teste seguro

No Codex, abra uma nova conversa/thread.

Use este prompt:

@Chrome pesquise 5 notícias recentes sobre agentes de IA no navegador.

Abra as fontes em abas separadas.

Crie um resumo em tópicos com:

- nome da ferramenta
- empresa
- o que ela faz
- risco de segurança ou privacidade

Não faça login em nenhum site.
Não envie formulários.
Não clique em anúncios.
Não baixe arquivos.

---

## 9. Teste com site próprio

Use este prompt:

@Chrome abra o site Café com Dados e Gatos.

Analise a página inicial como se fosse um visitante novo.

Liste melhorias em:

- clareza
- navegação
- chamada para ação
- aparência visual
- SEO
- velocidade percebida

Não altere nada no site.
Apenas analise e gere recomendações.

---

## 10. Teste com pesquisa de produtos

Use este prompt:

@Chrome pesquise notebooks na Shopee abaixo de R$ 3500.

Abra os 5 resultados mais relevantes.

Monte uma tabela com:

- nome do produto
- preço
- memória RAM
- armazenamento
- placa de vídeo, se tiver
- avaliação
- link

Não compre nada.
Não faça login.
Não adicione produtos ao carrinho.

---

## 11. Explicação para o vídeo

O Codex com Chrome permite que a IA trabalhe no navegador real.

Isso é diferente de um navegador interno simples porque ele pode usar:

- perfil real do Chrome
- abas reais
- sites onde o usuário já está logado
- cookies
- contexto de navegação
- grupos de abas

Por isso, a automação fica mais poderosa.

Mas também fica mais sensível.

A regra é simples:

Quanto mais acesso a IA tem, mais governança ela precisa.

---

## 12. Diferença entre navegador interno e extensão Chrome

Use o navegador interno do Codex para:

- páginas públicas
- localhost
- previews de site
- testes simples
- páginas sem login

Use a extensão Chrome para:

- sites com login
- ferramentas do dia a dia
- sistemas internos
- tarefas que dependem do perfil real do navegador

---

## 13. Boas práticas

Use tarefas pequenas.

Exemplo ruim:

Resolva toda a minha rotina de trabalho no navegador.

Exemplo bom:

Abra meu site e analise apenas a página inicial.

Exemplo bom:

Pesquise 5 ferramentas e monte uma tabela comparativa.

Exemplo bom:

Leia esta página e me diga os principais pontos.

---

## 14. Regras de segurança

Não permita ações automáticas em sites sensíveis.

Não deixe contas financeiras abertas.

Não deixe WhatsApp pessoal aberto.

Não deixe e-mail principal aberto.

Não salve senhas no perfil de teste.

Revise antes de aprovar ações.

Se o Codex tentar interagir com uma página errada, interrompa a tarefa.

---

## 15. Frase para explicar no vídeo

O Codex com Chrome não é só um chatbot olhando uma tela.

Ele começa a operar ferramentas reais, em um navegador real.

Isso aproxima a IA do trabalho prático do dia a dia.

Mas também exige isolamento, permissão e revisão humana.

---

## 16. Fechamento sugerido

A grande virada aqui não é só a IA abrir abas.

A virada é a IA trabalhar dentro do ambiente onde o trabalho real acontece.

Chrome, sites, sistemas, pesquisas, planilhas, formulários e ferramentas online.

Mas agente bom não é agente solto.

Agente bom é agente com limite, revisão e autorização.
